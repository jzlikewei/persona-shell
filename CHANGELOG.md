# Changelog

## 0.1.0 (2025)

Initial release.

- Feishu WebSocket 消息桥接
- Director 进程管理（named pipe IPC）
- FLUSH 上下文刷新机制
- DirectorPool 群聊多 Director 支持
- 消息队列（correlation ID + cancel）
- Web 控制台 + HTTP API（localhost:3000）
- 后台任务系统（SQLite 存储）
- Cron 调度器
- MessagingClient 抽象（支持飞书 / Web 双通道）
- launchd 服务化支持
